# x-image-downloader

Chrome/Edge extension for saving X/Twitter post images to a selected local folder.

## Development

```powershell
pnpm install
pnpm build
```

Load `dist/` from `chrome://extensions` with Developer mode enabled.

## MVP scope

- Save post images from `x.com` and `twitter.com`.
- Choose a save folder once from the extension options page.
- Save with one click from the hover button shown on images.
- Videos, GIFs, bulk save, and Chrome Web Store packaging are out of scope for the MVP.
